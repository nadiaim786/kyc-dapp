# 🔐 Blockchain-Based KYC Verification DApp

A full-stack decentralized KYC verification system using Ethereum blockchain.

## 🚀 Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MySQL
- Blockchain: Ethereum + Solidity + MetaMask

## ✨ Features

- 🦊 MetaMask wallet authentication
- 📩 OTP email verification
- 🔐 Secure KYC submission
- 🏦 Admin dashboard for verification
- ⛓️ Blockchain storage for verified users
- 📊 Real-time KYC status tracking

## 👤 User Flow

1. Connect MetaMask wallet
2. Fill KYC details
3. Receive OTP via email
4. Verify OTP
5. KYC stored securely

## 🏦 Admin Flow

1. Login with bank credentials
2. View KYC requests
3. Approve verified users
4. Store data on blockchain

## ⚙️ Setup Instructions

### Backend
cd backend
npm install
node server.js


### Frontend

cd frontend
npm install
npm run dev
